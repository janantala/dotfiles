# dotfiles

My dotfiles

## What is included?

### vim config
- vundle

### zsh shell config

#### Promt
- [pure promt](https://github.com/janantala/pure)

#### Exports
- android sdk
- texlive
- heroku

#### Functions
- gifify() - make .gif from .mov
- clone() - clone git repo, cd into it and open it in sublime text

## Configuration
Make symbolic links into your $HOME directory

```shell
.vim -> ~/.vim
.vimrc -> ~/.vimrc
.zsh -> ~/.zsh
.zshrc -> ~/.zshrc
```
